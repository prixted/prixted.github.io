프로젝트 메인 화면입니다.
매장안내 버튼을 누르면 나오는 화면입니다.
각 층에 어떤 매장이 있는지를 알려줍니다.		 매장 세부내용입니다. 지도, 로고, 매장위치, 전화번호,
운영시간등을 표시해 줍니다.
만약 매장이 음식점일 경우, 예약주문이 가능합니다. 기본적인 가격표와 수량을 정한 후 주문 가능합니다.
이벤트, 행사 안내입니다. 매장검색 화면입니다.
매장 이름과 카테고리별로 검색을 할 수 있습니다.
매장 선택후 선택매장 조회를 누르면 상세페이지로 이동합니다.
주차 선예약 기능입니다. 주차 예약 후 주차를 합니다.
주차 후 주차자리확인, 선결제 후 출차. 등 가능합니다.

mycode

제가 맡은 파트는 주차관련 기능 파트였습니다. 주요 기능은

  • 차량이 있을 경우 -> 우선예약 기능
  • 초록색 자리가 있을 경우 -> 차량번호, 이름, 전화번호로 예약
  • 차량을 주차한 후, 차량번호 or 전화번호로 주차위치 확인 가능(이름은 동명이인으로 제외)
  • 차량 주차위치 확인 후 뒤로가기 or 출차가능
  • 사전출차시 시간을 계산하여 요금계산, 15분(혹은 설정한 시간)이내로 출차를 요청하는 알림

주차데이터관리는 .dat파일을 만들어서 그 안에 쌓이도록 하였습니다.
또한, 주차관련 데이터가 변경될 때는 ObjectIntputStream("first.dat")파일을 불러온 후, 값 변경
그리고 그 값을 다시 ObjectOutputStream("first.dat")으로 저장합니다.		 주차 자리가 비어있나 확인 후
비어있으면 초록색, 차 있으면 붉은색으로 표시합니다.
비어있는 공간을 누르면 값 입력 후 , 저장합니다.
주차 자리를 찾아주는 메소드
전화번호 값, 차량번호 값 둘 중 하나만 들어오면 찾기 메소드 실행 만약 두개의 값이 모두 없으면 "잘못 입력하셨습니다" 출력
값을 입력받은 후, 값이 null이 아니고, 맞는 값이 있으면 true 반환
null값이 아니지만, 맞는 값이 없으면 "차량을 찾을 수 없습니다" 출력
 주차자리 확인 후, 혹은 주차요금 정산을 누르면
주차요금 정산 페이지로 이동
주차예약을 할 때 저장해놓은 시간과
현재 시간을 비교한 뒤에, 30분마다 1000원씩 부과
